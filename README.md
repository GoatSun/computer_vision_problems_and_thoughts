# computer_vision_problems_and_thoughts

1、现在多维度的backbone 的融合过程中能否利用queries进行feature融合？从而实现“实例-部件”级解耦即同时建模实例级结构（这个人是谁）和部件级语义（他的左手、右腿在哪），最后在根据具体任务需求来对每个人的具体区域或者人与人之间的区域进行分割（例如分割腋下区域、握手区域、挽臂区域等）
2、由于图像视野的限制，目前的CV网络很难识别出不完整的物体（部分区域在图像外侧），但人类视觉是可以通过想象视野外侧的内容识别出视野内不完整物体的类别或轮廓等，要如何让AI也能拥有“世界模型”（World Model）
